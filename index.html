Self Learning Bot

Developed a self-learning chatbot using Spacy and GPT-2 to handle tasks like answering questions, performing web searches, and providing language translations.

Created a virtual assistant chatbot named Friday that can answer questions, perform web searches, play YouTube content, and provide language translation services.

The bot uses Spacy for natural language processing and GPT-2 for generating intelligent responses.
