Movie Recommender System

Personalized movie recommendations using Python.

Collaborative Filtering

Collaborative Filtering is a recommendation technique that leverages the preferences and behaviors of multiple users to suggest new items. In this project, it was used to recommend movies by analyzing user ratings and drawing patterns from users with similar tastes. The key idea behind collaborative filtering is that if two users rate several movies similarly, the system can predict how one user would rate a movie the other has watched but not yet rated.

There are two main types of collaborative filtering used in this project:
User-Based Collaborative Filtering: This approach finds users who have rated movies in a similar way to the target user and recommends movies that those similar users have enjoyed.
Item-Based Collaborative Filtering: Here, the system finds similarities between movies based on user ratings and recommends movies that are rated similarly by users who liked a specific movie.
By focusing on user-item interactions, collaborative filtering generates highly personalized movie suggestions, improving the user experience by recommending content that aligns with users' past behavior.

Fuzzy String Matching

In the content-based recommendation system, fuzzy string matching was employed to compare movie titles and genres. It allowed the system to handle user input flexibly by matching similar or slightly misspelled movie titles. This ensured that even vague or partial inputs would return relevant movie recommendations.

TF-IDF was used to prioritize important terms in movie descriptions and genres. Matrix Factorization

Used within the collaborative filtering model, matrix factorization techniques like Singular Value Decomposition (SVD) helped decompose the large user-movie rating matrix into smaller, more manageable matrices. This improved the accuracy and performance of the recommender by discovering latent factors influencing movie preferences.
