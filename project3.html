Spotify Playlist Analytics Dashboard

The Spotify Playlist Analytics Dashboard is an interactive data visualization tool designed to provide insights into user listening habits and track performance metrics. Built using Power BI, this dashboard allows users to explore key data points such as:

  • Track Information: Displays the song title, artist, release date, number of streams, and other metadata.
  • Music Attributes: Key musical attributes such as danceability, energy, valence, speechiness, and liveness are highlighted, providing insights into the nature of the tracks.
  • Streams Analysis: The dashboard tracks and ranks songs based on their total streams, offering a comparison of popular tracks over time.
  • Yearly Performance: Users can filter and view data based on different years (from 2018 to 2023), observing trends in track performance across different time periods.
  • Energy Percentage: A dynamic gauge tracks the energy level of a song, helping users understand the intensity and feel of a track.
  • Top Song vs. Average: A performance indicator compares the highest streamed track with the average, giving a clear view of outliers.
  • Interactive Filters: Date range selectors, track names, and artist filters enhance the userâ€™s ability to explore data according to specific criteria.
  • Track by Release Date: A timeline chart visualizes when tracks were released, giving insights into music trends over the decades.

This project involved data preprocessing techniques such as handling missing values, normalizing data for consistency, and transforming it for visualization. The dashboard provides a comprehensive view of streaming data, making it a powerful tool for music analysts and users seeking to explore music trends and insights.

Technologies Used

  • Power BI for interactive data visualization and dashboard design.
  • Python for data preprocessing, cleaning, and transforming Spotify playlist data.
